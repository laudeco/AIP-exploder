# AIP explode

This tool explode the AIP file in one file per AIP category.
You can also reject all area above 2500ft.

## How to use
1. Download the project
2. run Compose
    ```
    $ composer install
    ```
3. Put your _*.aip_ file at root.
4. Adapt the run.php
    ```
    $ composer install
    ```
